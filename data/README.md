# Datos locales

Colocar aquí datos propios solo en una copia local privada. No subir XDF
crudos reales, videos, capturas ni logs identificables a GitHub.

La excepción es `synthetic/`, que contiene un XDF artificial sin datos
reales para probar los scripts públicos.

El XDF sintético no contiene audio digital, gaze ni sync probe; sólo sirve
como prueba mínima de EEG y marcadores.
